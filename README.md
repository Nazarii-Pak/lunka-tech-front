## Lunka Tech Test App

Simple planning screen that includes the following features:

- Drag & Drop Functionality
- Right-Click Context Menu
- Task Details Drawer
- Dynamic Data Handling

## Technologies used

* NextJS 15 (framework)
* TailwindCSS (style)
* HeadlessUI (UI)
* DnD Kit (Drag & Drop)
* Zustand (State Management)

## Project setup

1. Clone repo
2. `npm install`

## Running

##### run as develop mode

```
npm run dev
```


##### run as production mode

```
npm run build
npm start
```
